# Python HTTP Module Importer

## Описание
Система для импорта Python-модулей напрямую из HTTP/HTTPS URL. Позволяет загружать и исполнять код с удаленных серверов как локальные модули.

## Особенности
- Импорт обычных модулей (.py файлов)
- Импорт пакетов (директорий с __init__.py)
- Импорт подмодулей внутри пакетов
- Поддержка всех стандартных способов импорта
- Кэширование URL для производительности
- Обработка ошибок сети и HTTP

## Требования
- Python 3.6+
- Библиотека requests

## Установка
```bash
pip install requests
```

## Структура проекта
```
P5LR1/
├── Script.py              # Основной скрипт импорта
├── Server/                # Серверная часть
│   ├── remotemodule.py    # Удаленный модуль
│   └── mypackage/         # Пакет
│       ├── __init__.py
│       └── submodule.py
└── README.md
```

## Использование

### 1. Запуск сервера
```bash
cd Server
python -m http.server 8000
```
